# AGENTS.md

This is a learning/playground repo (`dreamquster/learn-claude-code`) for experimenting with OpenCode (Claude Code).

- **State:** Minimal Python project exploring LangChain + DeepSeek.
- **`.gitignore`:** Standard Python template with extras for Ruff, Marimo, Streamlit.
- **Scripts:**
  - `chat.py` — basic LangChain + DeepSeek chat via `langchain-openai`
  - Set `DEEPSEEK_API_KEY` env var before running
- **Commands:** `pip install -r requirements.txt && python chat.py`
